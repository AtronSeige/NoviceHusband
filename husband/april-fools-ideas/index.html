April Fools day can be so much fun, but I keep forgetting the actual tricks when it comes to the day. So here are some tricks. Some are more friendly than others.

Most of these are IT related.

General

  • Set everyone’s background wallpaper to BSOD or Blue Screen Of Happiness.
  • Install Clippy. Make sure you get a version that IT has approved of. There are infected versions.

Does Not Compile - Screwing with the devs

  • Change font in dev tools to Comic Sans
    • VS
    • VSCODE

Slightly nasty

  • If someone sits on a dark chair, you can easily pour a bit on water on the chair. It is nearly invisible and an uncomfortable wet seat for the person.
